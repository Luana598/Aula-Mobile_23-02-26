# Aula Mobile - 23/02/26

Este repositório contém códigos e exercícios desenvolvidos durante a aula de **Desenvolvimento Mobile** realizada em 23/02/26.

##  Objetivo

Praticar conceitos básicos de desenvolvimento para dispositivos móveis, incluindo estrutura de projeto, lógica de programação e implementação de funcionalidades simples em aplicativos.

## Tecnologias utilizadas

* Kotlin
* Android Studio IDE

## Como executar

1. Clone este repositório:

```bash
git clone https://github.com/Luana598/Aula-Mobile_23-02-26.git
```

2. Abra o projeto na IDE utilizada para desenvolvimento (Android Studio).

3. Execute o projeto em um emulador ou dispositivo físico.

## Aprendizado

Este projeto foi desenvolvido com fins educacionais para praticar conceitos iniciais de desenvolvimento de aplicativos móveis.
